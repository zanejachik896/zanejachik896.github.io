おすすめポイント ■3端子対応指紋認証USBメモリ
指紋暗号化USBドライブ。生体認証技術と安全なストレージ スペースを備えたフラッシュドライブで、データ紛失や 盗難にも大切なデータの漏洩を防ぎます。 マルチデバイスに対応した3ポートデザインの指紋認証USBメモリ。 Lightning（ライトニング）、USB2.0、Type-Cポートに対応 １台３役であらゆるニーズに対応し、データを保護します。 ※Type-CデバイスはOTG機能に対応している必要があります。

■指紋認証で大切なデータを保護
近年多発しているインターネット犯罪、データ流出も指紋認証ロックで 大切なデータを強固に保護。あなた以外のアクセスを遮断します。 フルディスク暗号化により、簡単に破られないデータセキュリティシステム プライバシーを保護し、プライベートなライフデータの漏洩を防いでくれます。

■360°全方向からスムーズに指紋を認証
あらかじめ指紋、パスワードを登録しておくことで 本体中央をタッチするだけでロックを解除します。 360°あらゆる角度からでもスムーズに認証可能

■高速認識＆高速転送
高速&安定した書き込み読み込みスピードを実現。指紋認証はたったの1秒でロックを解除します。

■最大64人分 ID登録が可能
組織やチームでの作業時に、最大64人分のパートナーの指紋IDを入力可能 重要な仕事の機密を守り、権限のない人のアクセスを防ぎ、データの流出を防ぎます。 （指紋は後からでも追加登録可能です。）

■ポータブルで持ち運びに最適
小型で軽量なポータブルサイズなので外のへ持ち運びに便利です！ もしも外出先で紛失しても指紋認証でデータを保護します。

ご注意 ※初めて使用する際は、iOSデバイスの「APP Store」よりソフトウェア「Mobile memory HD」をダウンロードする必要があります。
※USBフラッシュディスクに登録する指紋とパスワードは、iOS端末の「Mobile memory HD」ソフトウェア上でのみ追加が可能です。
※メモリにパスワードと指紋が追加される前、コンピュータはUSBフラッシュディスクを読み込むことが可能ですが、パスワードと指紋を設定した後は、指紋ロックが解除された場合のみ、USBフラッシュディスクを認識することができます。
※Type-Cインターフェイスを使用する場合、iPhoneでパスワードと指紋が追加される前であれば、Android携帯でもUSBフラッシュドライブを直接読み取ることができます。ただし、パスワードと指紋を設定した後は、指紋によるロック解除をした後にのみ、Android携帯でもUSBフラッシュドライブを認識することができます。
※指紋認証とパスワードの設定は、LightningおよびType-Cポートを搭載したiOSデバイスにのみに対応し、その他のシステム設定やパスワードには対応していませんので、ご了承ください。
※パスワード、指紋が紛失した場合はデータにアクセスできませんのでご注意ください。
