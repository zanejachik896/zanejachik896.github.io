THE NORTH FACE ザ・ノース・フェイス RED RUN LONG PANT レッドランロングパンツ NY82395 / メンズ / ランニング

製 品 説 明


ランニング時の運動性能を損なわず、動きやすさと効率的な保温性を両立した高機能保温パンツです。
防風性と耐摩耗性に優れるパーテックスカンタムをメイン素材に使用し、軽量で保温性が高い「PRIMALOFTBIO」をボール状に集積させた「THERMOBALLBIO」を腰まわりと太ももに封入。
嵩張ることがなく、効果的な保温力を発揮します。
さらに、中わたのTHERMOBALLBIOは、生分解性をもつ100％リサイクル材で、環境にも配慮した素材です。
多少の雪や雨に対応するはっ水加工済み。
後ろ腰中央に、スマートフォンサイズの小物が収納できるファスナーポケットを配備しています。
同シリーズの「Red Run Hoodie（NY82393）」や「Red Run Vest（NY82394）」とセットアップが可能です。

■Fabric : ＜表地＞PERTEX QUANTUM ECO 20D Ripstop（ナイロン100％）＜中わた＞THERMOBALL BIO（ポリエステル100％）＜股下・膝下＞Versa Active ECO（ナイロン50％、ポリエステル45％、ポリウレタン5％）＜裏地＞Recycled Nylon Taffeta（ナイロン100％）

■Function : 生分解性中わたTHERMOBALLBIOを採用／左右脇ポケット仕様／後腰中央ファスナー付ポケット／ウエストスピンドル仕様／リフレクターロゴ

■原産国 : ベトナム
