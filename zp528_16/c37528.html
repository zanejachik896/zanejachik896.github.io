商品説明【商品概要】
プロセッサ: Microbit V1 は 16MHz ARM Cortex-M0 プロセッサを搭載した Nordic nRF51822 システムオンチップ (SoC) を搭載しており、Microbit V2 は 64MHz ARM Cortex-M4 プロセッサを搭載した Nordic nRF52833 SoC を搭載しています。 Microbit V2 のアップグレードされたプロセッサにより、パフォーマンスと機能が向上しました。
メモリ: Microbit V1 には 16KB の RAM と 256KB のフラッシュ メモリが搭載されていますが、Microbit V2 には 128KB の RAM と 512KB のフラッシュ メモリが搭載されています。 Microbit V2 のメモリの増加により、より複雑なプログラムとより大きなストレージ容量が可能になります。
電源管理: Microbit V2 には強化された電源管理機能が含まれており、より効率的なエネルギー使用とバッテリ寿命の延長が可能になります。 また、電源オプションのより広い電圧範囲もサポートします。
ディスプレイ: Microbit V1 と V2 はどちらも 5x5 LED マトリックス ディスプレイを備えていますが、Microbit V2 のディスプレイはより明るく鮮明で、さまざまな照明条件でも視認性が向上しています。
センサー: Microbit V2 には、Microbit V1 と比較して追加のセンサーが搭載されています。 マイクとスピーカーが含まれており、サウンド関連のプロジェクトやアプリケーションが可能になります。
