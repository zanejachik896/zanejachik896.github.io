【ラウンドフラッシュヘッド専用】高度な光制御アクセサリでラウンドフラッシュゲームを強化し、特に Z1-C/Z1-N/Z1-SおよびZ2-C/Z2-N/Z2-Sラウンドヘッドフラッシュに対応します。Godox V1 フラッシュ AK-R1 アクセサリーセットやその他のラウンドヘッドフラッシュに対応します。スクエアヘッドフラッシュを備えた CRM2 アクセサリを使用するには、 SFR1 アダプタを別途購入してください。
【クリエイティブな照明ををあなたの手元に】ラウンドヘッドフラッシュを最大限に活用します。円錐形のスヌートがビームを正確に狭めて被写体を強調し、ハニカムグリッドがコントラストを高めて輪郭を強調し、ドームディフューザーと拡散パネルが光を散乱させて柔らかくして色調を調和させ、バウンスディフューザーが影を埋めてキャッチライトを追加し、4ウェイバーンドアが照明パターンを作成し、3色のカラージェルがさまざまなテーマに合わせて色合いを追加します。
【マグネットで簡単取り付け】これらのアクセサリはマグネットリングでしっかりと取り付けられ、簡単に交換して迅速な調整が可能です。単独で使用することも、重ねて使用することもでき、ユニークな効果を得ることができます。アダプターフレームとリングが付属しており、反射ディフューザーやカラージェルををシームレスに取り付けることができます。
【軽量＆ポータブル】PC、PETなどの耐久性に優れた軽量プラスチックで作られており、総重量はわずか290gと荷物にかさばらず、外出先での撮影に便利です。強化シェルとリストストラップを備えた防水キャリングケースで、保管と持ち運びが簡単です。
【パッケージ内容】 1 x 4ウェイバーンドア、1 x 180°ドームディフューザー、6 x カラージェル、1 x ハニカムグリッド、1 x 拡散パネル、1 x 円錐スヌート、1 x バウンスディフューザー、1 x アダプターフレーム、1 x アダプターリング、、光の究極の制御と形状を実現します。
